# Changelog

## 1.0.0 (2025)

- Initial public release
- Terminal AI chat with OpenTUI + React
- Plan and Build modes
- Multi-model support (Anthropic, OpenAI, Google, Groq, OpenRouter)
- Persistent sessions via Prisma + SQLite/PostgreSQL
- Clerk OAuth authentication
- Polar.sh usage billing
- MCP server for agent tool execution
- Local tool execution (read, write, edit, bash, glob, grep)
- Skill system for extensible workflows
- Project indexing and context management
